web-boardimage
==============

An HTTP service that renders chess board images.

Usage
-----

```
python3 server.py [--port 8080] [--bind 127.0.0.1]
```

Installation
------------

Requires Python 3.4+.

```
sudo apt-get install python3-dev libffi-dev libxml2-dev libxslt1-dev

pip install -r requirements.txt
```
